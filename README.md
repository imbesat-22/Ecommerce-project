# 🛒 Ecommerce Project Front-End Development

A responsive Ecommerce web page built with HTML and CSS as part of my Internship porject.

---

## 📁 Project Structure
 
```
Ecommerce-project/
├── index.html       # Main HTML page
├── styles.css       # All styling
├── images/          # Images and assets
└── README.md        # Project documentation
```
 
---

 ## ✨ Features
 
- Responsive navbar with search bar and cart icons
- Hero banner section with category sidebar
- Deals and offers section
- Product grid layout
- Newsletter subscription section
- Footer with links and app store buttons
---

## 🛠 Built With
 
- HTML5
- CSS3 (Flexbox and Grid)
- No frameworks or libraries — pure HTML and CSS
---
 
## 📚 What I Learned
 
- Semantic HTML structure
- CSS Flexbox for navbar and card layouts
- CSS Grid for multi-column product grids
- Responsive design principles
- Git and GitHub workflow
---

## 🚀 How to Run Locally
 
1. Clone the repository:
   ```bash
   git clone https://github.com/imbesat-22/Ecommerce-project.git
   ```
2. Open the project folder in VS Code
3. Right-click `index.html` → Open with Live Server
